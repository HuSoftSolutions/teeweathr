TeeWeathr — embed tester

Live preview of every meaningful widget variant. Each card has a copy-paste <iframe> ready for Wix or any other site builder. Snippets are auto-filled with this page’s URL — no manual editing.

Host: —
To test on Wix:
  1. Make sure the dev server is running (npm run dev).
  2. Expose it publicly: ngrok http 3000 (or cloudflared tunnel --url http://localhost:3000, or push a branch for a Vercel preview URL).
  3. Open this same page on the public URL — e.g. https://your-tunnel.ngrok.io/embed-tester.html — so the snippets below auto-fill with that host.
  4. In Wix: Add → Embed Code → Custom Embeds → Embed HTML. Click Copy snippet below and paste it into the embed dialog.

Default demo course is Pebble Beach Golf Links. Change lat, lon, name, holes, par in the snippet to point at any course. Available URL params: theme (dark|light), accent (default|blue|purple|red|orange|zinc), branding (true|false), ads (true|false). When using a real key, all of these become server-controlled and URL params are ignored.
